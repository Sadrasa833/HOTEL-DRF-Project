# 🏨 East Hotel Management System

یک سیستم جامع مدیریت هتل که با معماری مدرن و مقیاس‌پذیر برای مدیریتِ فرآیندهای رزرو، حسابداری و ویترینِ هتل طراحی شده است. تمرکز این پروژه بر روی **امنیتِ داده‌های مالی** و **تجربه‌ی کاربری (UX) لوکس** بوده است.

---

## 🏗️ معماری فنی (Technical Architecture)

این پروژه از یک ساختار **Decoupled** استفاده می‌کند تا عملکرد و امنیت به حداکثر برسد:

### ⚙️ بک‌اِند (Django + DRF)
- **RESTful API:** پیاده‌سازی بیش از ۱۵ اِندپوینتِ بهینه‌شده برای مدیریتِ اتاق‌ها، صورت‌حساب‌ها و منوی رستوران.
- **تضمینِ تمامیتِ داده‌ها:** استفاده از منطقِ سمت سرور برای جلوگیری از `Overbooking` و تداخل در تاریخ‌های رزرو.
- **ماشین‌حساب مالی خودکار:** محاسبه‌ی خودکارِ هزینه‌ی اقامت، مالیات (VAT)، تخفیف‌ها و کسر بیعانه با استفاده از کلاس‌های محاسباتی در جنگو.
- **احراز هویت امن:** پیاده‌سازی JWT (JSON Web Token) برای مدیریت نشست‌های کاربری.

### 🎨 فرانت‌اِند (React.js)
- **Dark Luxury UI:** طراحیِ اختصاصی با الهام از استانداردهایِ هتل‌های بوتیک (تمِ مشکی و طلایی).
- **Interactive UX:** استفاده از گالری‌های لایت‌باکس، انیمیشن‌های پارالکس (Parallax) و چیدمانِ زیگ‌زاگی (Z-Pattern).
- **Responsive Design:** طراحیِ استاندارد برای نمایش در انواع مانیتورها.

---

## 📦 قابلیت‌های کلیدی (Key Features)

- [x] **پنل مدیریت پیشرفته:** ادمین می‌تواند به صورت داینامیک منوی رستوران و بار را تغییر دهد.
- [x] **سیستم فاکتوردهی:** مشاهده آنلاین صورت‌حساب‌ها با تفکیک هزینه‌های اقامت و خدمات.
- [x] **وضعیت‌های رزرو:** جریانِ کاملِ `Pending` -> `Confirmed` -> `Checked-out`.
- [x] **تجربه‌ی لوکس:** استفاده از فونت استاندارد فارسی (وزیرمتن) و المان‌های طراحیِ نوین.
